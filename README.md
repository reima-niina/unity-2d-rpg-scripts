# unity-2d-rpg-scripts
Unity(C#)で個人制作した2DRPGのスクリプト集です。
<実装>基本システム・進行まわり/Al連携NPC/誓約システム/バトルシステム（ターン制）/ステータス・レベルアップ/仲間システム/フィールドの敵・エンカウント/ボス戦・エンディングフロー/カットシーン・会話システム/回復システム/UIまわり/BGM・SEシステム
<主要ファイル>BattleManager.cs/OathSkillExample.cs/ScenarioManager.cs/BGMManager.csなど
<工夫>誓約システムや仲間の死と復活の演出、場所ごとのBGM、ChatGPTと連携したNPCが主な工夫点であり、制作過程ではログ出力と再現手順の整理で原因切り分け、機能単位で分割し、デバック性の改善を行った。
<制作範囲>企画～UI/演出～実装まで個人で担当。
